Getting started

  1. Install and start Elasticsearch server.
    2. sudo dpkg -i elasticsearch-<version>.deb
sudo service elasticsearch start
  2. Install and setup Kibana. Require a web server.
    3. tar -zxvf kibana-<version>.tar.gz \
	--strip-components=1 \
	-C <webserver-path>/kibana

    Note: you may need to edit 'config.js' file and set the elasticsearch address.

    Example: elasticsearch = http://127.0.0.1:9200

  3. Install Haka and Hakabana 
    4. sudo dpkg -i haka-<version>.deb
sudo dpkg -i hakabana-<version>.deb
  1. Setup and start Haka.
    2. haka -c /usr/share/haka/hakabana/haka.conf

    Note: you may need to update elasticsearch setting in /usr/local/share/haka/hakabana/config.lua

  2. Visit kibana webpage at http://<webserver-address>/kibana.

  3. Load Kibana dashboard.

  • Building from source ?

  • Going further: All data are exposed to hakabana through security rules in /usr/local/share/haka/modules/misc/hakabana/. So feel free to customize these rules to fit you own goals.
